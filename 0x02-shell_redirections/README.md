# 0x02. Shell, I/O Redirections and Filters

This project covers:
- Output redirection (`>`, `>>`)
- Input redirection (`<`)
- Pipes (`|`)
- Filters such as `head`, `tail`, `cut`, `grep`, `sort`, `uniq`, `wc`
- File descriptors (stdin, stdout, stderr)
- Creating and manipulating files using shell commands

Each task is implemented as an executable Bash script.

## Requirements
- Scripts must start with `#!/bin/bash`
- Scripts must be executable
- No using `sed` unless specified
- All files follow ALX style guidelines
